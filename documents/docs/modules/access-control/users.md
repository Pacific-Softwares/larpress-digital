# Users

Manage user accounts for the admin panel and frontend.

## Overview
- Create, edit, deactivate users
- Assign roles and direct permissions
- Reset passwords and enforce MFA (if enabled)

## Typical actions
1. Create a user and set a temporary password
2. Assign appropriate role(s)
3. Optionally grant specific permissions
