# Communication Settings

Configure Twilio and SMTP credentials, then send test messages without leaving the page.

## Sections
- **Twilio SMS Settings**: SID, token, from number, optional test recipient/message, and “Send Test SMS” action
- **Email Settings**: SMTP host/port/username/password, from address, optional test recipient/message, and “Send Test Email” action
- **Save Settings** button persists credentials while ignoring one-off test fields

## Tips
- Use dedicated test numbers/emails before going live
- If tests fail, the page surfaces error messages via notifications
- Credentials are stored centrally for all outbound communications
