## Quick Start

Install LarPress in minutes:

```bash
# Clone the repository
git clone https://github.com/yourusername/larpress.git

# Install dependencies
composer install
npm install && npm run build

# Run web installer
php artisan serve

# Visit http://localhost:8000/install
```
